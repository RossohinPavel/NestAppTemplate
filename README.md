# NestAppTemplate

Шаблон NestJs приложения под разные нужды.


## Стэк

- pnpm - Используется этот менеджер пактов.
- typescript - Само собой, последней версии.
- Nest - Также, последней версии.
- Fastify адаптер. Стандартное ядро Nest заменено на `Fastify` более быструю альтернативу.


# Использование

Предполагается, что эту репу форкнут и/или будут использовать как шаблон для создания рабочих приложух.
Соответственно, стянуть проект, написать `pnpm start:dev` и на `localhost:3000` увидеть `Hello world!`.


# Примечания

- Приложение жестко запускается на `0.0.0.0:3000`. Сделано это для удобства работы с докером.
Если такой расклад не устраивает, вы всегда можете поменять это на свое усмотрение.
